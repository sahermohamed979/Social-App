# Social App

A modern, full-featured social media web application built with React and Vite. This project demonstrates advanced React development, state management, and UI/UX skills in a Facebook-inspired interface.

---

## 🚀 Why This Project?

This project is designed to showcase:

- **Modern React (v19) with Hooks and Context**
- **Type-safe, modular, and scalable code structure**
- **Real-world features:** authentication, protected routes, CRUD operations, profile management, notifications, and more
- **Best practices:** code splitting, reusable components, custom hooks, and state management
- **Responsive, accessible, and SEO-friendly UI**
- **Professional tooling:** ESLint, Tailwind CSS, React Query, React Hook Form, Framer Motion, and more

---

## ✨ Features

- User authentication (register, login, logout)
- Protected routes for authenticated users
- Responsive layout with left/right sidebars and top navigation
- Create, update, delete posts and comments
- Like and react to posts/comments
- Profile management (update info, change password, upload profile picture)
- Real-time data fetching and caching with React Query
- Toast notifications and modals for user feedback
- Facebook-like UI/UX
- SEO support with `react-helmet`
- Accessibility and mobile-first design

---

## 🛠️ Technologies Used

- **React** (v19) – UI library
- **Vite** – Fast build tool and dev server
- **React Router** – Routing and navigation
- **React Query** – Data fetching, caching, and state management
- **Axios** – HTTP requests
- **Tailwind CSS** – Utility-first CSS framework
- **Flowbite** – UI components for Tailwind
- **Framer Motion** – Animations
- **React Icons** & **FontAwesome** – Icon libraries
- **React Hook Form** – Form management and validation
- **SweetAlert2** & **React Toastify** – Alerts and notifications
- **ESLint** – Linting and code quality
- **react-helmet** – SEO and meta tags
